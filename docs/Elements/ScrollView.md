## ScrollView

An element that displays content inside a scrollable area.

#### Fields

| Field   | Type           | Description                                    |
| ------------------------------------------------------------------------- |
| ScrollOffset     | Vector2        | Current scrolling position            |

#### Events

| Event                   | Description                                     |
| ------------------------------------------------------------------------- |
| ScrollEvent             | Executed whenever the `ScrollOffset` changes    |
